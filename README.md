# ごはん日記（gohan_nikki)

## サイト概要
### サイトテーマ
ユーザー同士でメニューを公開し合い、家庭料理の情報交換をする場所としてのアプリ
また、その日に作った料理をカレンダーに保存して、いつに何を作ったのかを保存する日記をつけることができる。

### テーマを選んだ理由
自分自身が料理をすることが好きで、いろんな料理の作り方を知りたいし、それをみんなに知ってほしい。

### ターゲットユーザ
料理好き・主婦・一人暮らしの方

### 主な利用シーン
ご飯を作るときや、献立を考えているとき。
